# Applicant Tracking System (ATS)

Webbasiertes Bewerbermanagementsystem, entwickelt im Rahmen des Moduls DLBITPEWP01 – Einstieg in die Web-Programmierung.

## Verwendete Technologien

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- H2 Database
- Thymeleaf
- Spring Security
- Bootstrap
- Chart.js

## Anwendung starten

```bash
mvn spring-boot:run
```

Anschließend im Browser öffnen:

http://localhost:8080

## Testzugang Recruiter-Dashboard

Benutzername: recruiter

Passwort: password

Login:

http://localhost:8080/login

## Funktionen

- Stellenübersicht
- Stellendetails
- Online-Bewerbung
- Lebenslauf-Upload
- Recruiter-Dashboard
- Suchfunktion
- Statusverwaltung
- Statistische Auswertung
